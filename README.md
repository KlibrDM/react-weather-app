# react-weather-app
Live preview: https://klibrdm.github.io/react-weather-app/

### Weather App made with ReactJS

## Setup
* Open a terminal of your choice
* `git clone https://github.com/KlibrDM/react-weather-app.git`
* `cd react-weather-app`
* `npm install`
* `npm start`
